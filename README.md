# hospital-emergency-room-dashboard

# Overview
This project focuses on analyzing hospital emergency room performance using Microsoft Excel. The dashboard provides insights into patient volume, average wait time, and patient satisfaction to support better operational decision-making.

# Objective
- Analyze patient flow in the emergency room
- Monitor average wait time
- Evaluate patient satisfaction levels
- Identify patterns to improve hospital efficiency

# Dataset
The dataset used in this project contains hospital emergency room records with the following fields:

- Patient ID – Unique identifier for each patient  
- Admission Date – Date of patient entry into the emergency room  
- Patient Name – Name of the patient  
- Gender – Gender of the patient  
- Race – Patient demographic information  
- Department – Department handling the case  
- Referral – Source of patient referral  
- Admission Status – Indicates whether the patient was admitted or not  
- Satisfaction Score – Patient satisfaction rating  
- Wait Time – Time taken before the patient was attended  

(Note: This dataset is used for learning and analysis purposes.)

# Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Slicers for interactivity
- Drill-down analysis
- Charts and KPIs

# Dashboard Features
- Interactive slicers to filter data
- KPIs: Total Patients, Average Wait Time, Satisfaction Score
- Visual charts for trend analysis
- Drill-down functionality for detailed insights

# Key Insights
- Peak patient load observed during specific time periods
- Higher wait times correlated with increased patient volume
- Patient satisfaction decreases as wait time increases

# How to Use
1. Download the CSV file
2. Open in Microsoft Excel
3. Use slicers to filter data
4. Click on charts to explore drill-down insights
